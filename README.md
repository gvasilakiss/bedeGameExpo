# BedeExpo

A simple React Native app showcasing a list of games for the Bede Expo test event. Each game has a title and a URL to play it.

## Tech Stack
- React Native
- TypeScript
- Expo


## Getting Started
1. Clone the repository:
    ```bash
    git clone https://github.com/gvasilakiss/bedeGameExpo
    ```
2. Navigate to the project directory:
    ```bash
    cd bedeGameExpo
    ```
3. Install dependencies:
    ```bash
    npm install or bun install
    ```
4. Start the Expo development server:
    ```bash
    npm start or bun start
    ```
5. Follow the instructions in the terminal to run the app on an emulator or a physical device using Expo Go or scanning the QR code.
